# Changelog

All notable changes to this project are documented in this file.

## [1.0.2] - 2026-03-09

### Added

- Added new subcategories: `Rich Text & Markdown`, `Local-First Databases & Sync`, and `ORM & Query Builders`.
- Added 12 resources across UX, State & Data, Forms, Device APIs, and Performance sections.

### Changed

- Removed `Dripsy` and two Expo Router docs resources from the index.

## [1.0.1] - 2026-03-07

### Added

- Added new `MCP Integrations` category with `MCP Setup Guides` subcategory.
- Added MCP setup resources for Expo, Mixpanel, RevenueCat, and Supabase.
- Added `React Native Localize`, `Notifee`, `React Native Compressor`, and `React Native Fast Image` resources.

### Changed

- Removed `Expo Default Template` from starter resources.

## [1.0.0] - 2026-03-06

### Added

- Initial public release of the curated Expo and React Native DX resource collection.
- Automated schema validation, staleness checks, README generation, link checks, and Markdown linting.
- Deterministic README generation directly from `data/resources.yaml`.
